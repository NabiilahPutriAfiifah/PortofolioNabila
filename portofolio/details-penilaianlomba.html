Portofolio Sistem Pendukung Keputusan Penilaian Lomba Menggunakan Metode AHP - MAUT

Sistem Pendukung Keputusan Penilaian Lomba – Web App berbasis AHP & MAUT untuk meningkatkan efisiensi, akurasi, dan transparansi penilaian lomba HIMATIK.

Sistem Pendukung Keputusan Penilaian Lomba Menggunakan Metode Analytical Hierarchy Process (AHP) dan Multi Attribute Utility Theory (MAUT)

Proyek ini merupakan skripsi D4 Teknik Informatika yang berfokus pada pengembangan sistem pendukung keputusan (SPK) berbasis web untuk mendukung proses penilaian lomba di HIMATIK Politeknik Negeri Jakarta. Sistem ini dirancang untuk mengatasi keterbatasan penggunaan spreadsheet yang rawan error, kurang efisien, dan minim transparansi.

Aplikasi dikembangkan menggunakan Laravel (PHP), MySQL, serta HTML, CSS, dan JavaScript sebagai front-end. Proses perancangan antarmuka dibuat dengan Figma, dan pengembangan mengikuti metodologi Waterfall.

  • Metode AHP digunakan untuk menentukan bobot kriteria penilaian.
  • Metode MAUT digunakan untuk menghitung dan meranking nilai peserta.

Pengujian dilakukan dengan black box testing yang menunjukkan sistem berfungsi 100% sesuai harapan. Evaluasi usability menghasilkan skor SUS = 89,19 (Excellent) dan NPS = 67,5% (sangat baik), membuktikan sistem ini diterima dengan baik oleh pengguna.

Sebagai peneliti sekaligus pengembang utama, saya bertanggung jawab penuh mulai dari analisis kebutuhan, perancangan UI/UX, hingga implementasi dan pengujian sistem.

Sistem ini berhasil meningkatkan efisiensi, akurasi, dan transparansi penilaian lomba, sekaligus mendukung upaya digitalisasi kegiatan organisasi mahasiswa HIMATIK.
