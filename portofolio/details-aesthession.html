Aesthession – Online Fashion Store (Bytesfest 2022)

Proyek ini merupakan website e-commerce sederhana yang dirancang untuk menampilkan berbagai produk fashion lokal dan internasional. Website dibuat dengan teknologi dasar HTML, CSS, dan JavaScript (native), tanpa framework, sebagai bagian dari kompetisi Bytesfest 2022.

Fitur Utama :

  • Halaman utama dengan hero banner dan tagline brand
  • Halaman About berisi profil singkat toko
  • Kategori produk (pria, wanita, anak)
  • Shop page dengan daftar produk, harga, dan diskon
  • Sign In page untuk simulasi login pengguna
  • Desain responsif sederhana agar nyaman diakses di berbagai perangkat

Dalam proyek ini, saya berfokus pada frontend development. Tugas saya mencakup menyusun struktur halaman menggunakan HTML, membuat styling tampilan dengan CSS, serta menambahkan interaktivitas dasar menggunakan JavaScript, seperti navigasi antarhalaman dan tampilan produk. Selain itu, saya juga bekerja sama dengan tim untuk menyatukan elemen desain dan konten agar website dapat selesai sesuai kebutuhan lomba.
