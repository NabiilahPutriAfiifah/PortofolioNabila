Smart Locker – Web Application

Smart Locker adalah aplikasi web yang memungkinkan pengguna untuk membuka dan mengelola loker pintar hanya dengan masuk menggunakan kredensial. Setelah penggunaan selesai, pengguna juga dapat memberikan ulasan mengenai produk ini. Aplikasi ini dirancang dengan antarmuka yang sederhana dan user-friendly sehingga mudah dipahami oleh berbagai kalangan pengguna.

Fitur Utama

  • Login & autentikasi pengguna untuk akses loker
  • Dashboard untuk membuka dan mengelola loker pintar
  • Review system agar pengguna bisa memberi ulasan setelah penggunaan
  • UI sederhana & intuitif untuk memudahkan interaksi

Dalam proyek ini, saya berperan sebagai UI/UX Designer sekaligus Frontend Developer. Saya bertanggung jawab membuat desain antarmuka di Figma, mulai dari wireframe hingga high-fidelity prototype, dengan fokus pada kesederhanaan dan kemudahan penggunaan. Selain itu, saya juga membantu mengimplementasikan desain tersebut ke dalam bentuk aplikasi web melalui pengembangan frontend.
